# FPGA-CUBE
使用ZYBO Z2的PL端的FPGA实现了一个三阶虚拟魔方
通过Pmod扩展JE端外接六个独立按键控制魔方六个面的旋转，通过两个独立按键控制魔方旋转的角度
将魔方的立体图和展开图通过HDMI接口显示到显示器上
